Mental Health Treatment Predictor
Live SHAP · XAI

Trained and compared 5 machine learning models (Logistic Regression, Decision Tree, Random Forest, Gradient Boosting, KNN) on 292,000+ records to predict mental health treatment likelihood. Gradient Boosting achieved 71.5% accuracy and 0.773 ROC-AUC. Integrated SHAP explainability to surface per-prediction feature contributions.

Gesture-Controlled Chord Player
Live Computer Vision Web App

A browser-based instrument that uses real-time hand gesture recognition to trigger musical chords. Make a hand sign in front of your webcam and hear a chord play instantly — no installation required. Built in three versions: Python local tools, then a fully deployable web app using MediaPipe JS and Tone.js. Features a glowing hand skeleton visualizer, live waveform display, and mixed mic + chord recording.
